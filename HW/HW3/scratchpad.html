What is CSS

CSS holds the formating information for HTML files

    CSS Notes [A list]

    [Non list item]
  • each CSS rule has a selector and one or more declarations:
    h1 {color:blue; font-size:12px;}
  • Declarations consist of property-value pairs
  • CSS comments begin with "/*" and end with */

id and class selector

The id selector is used to specify a style for a single, unique element, using its id attribute.
#para1, selects the HTML element with the id ="para1"
 The class selector specifies the styling and formating for all classes associated with a class.
Specify all elements of a class with the ".CLASS_NAME" and elements of a type of some class with "TYPE.CLASS_NAME"
